# 🚇 METRO RUSH

> Run. Dodge. Survive.

A futuristic endless runner game inspired by fast-paced arcade gameplay.  
Collect coins, avoid obstacles, unlock characters, and survive as long as possible in the neon metro world.

---

## 🎮 Game Overview

**Metro Rush** is a single-player endless runner game built using pure HTML, CSS, and JavaScript.

The player runs automatically through a futuristic metro city while avoiding trains, barriers, tunnels, and other obstacles.

---

## ✨ Features

- 🏃 Smooth endless runner gameplay
- ↔️ Swipe / keyboard lane controls
- ⬆️ Jump and ⬇️ slide mechanics
- 🪙 Coin collection system
- 🚀 Multiple power-ups
- 👤 Unlockable characters
- 🏆 Leaderboard system
- 🎯 Missions & achievements
- 🌙 Dynamic day/night cycle
- 💾 Local storage save system
- 📱 Mobile friendly controls
- ⚡ Optimized smooth animations

# 🕹 Controls

| Action | Control |
|---|---|
| Move Left | ← Arrow / Swipe Left |
| Move Right | → Arrow / Swipe Right |
| Jump | ↑ Arrow / Swipe Up |
| Slide | ↓ Arrow / Swipe Down |
| Pause | P / ESC |
